🚀 Тестовая страница LiveChat

Эта страница демонстрирует работу виджета чата. Виджет подключается как script.

Как это работает

1. Запустите backend: npm run dev:backend

2. Инициализируйте БД: cd backend && npx prisma db push

3. Зарегистрируйте оператора и создайте проект через Dashboard

4. Подставьте projectId ниже (из созданного проекта)

5. Нажмите на кнопку чата в правом нижнем углу
