A simple LiveChat widget integration for WordPress. This plugin simply adds the LiveChat tracking script and configures your project ID.

== Description ==

The LiveChat Widget plugin offers an easy way to integrate your LiveChat system into any WordPress website. By installing this plugin, you can add the live chat widget to the bottom right of your site for visitors to contact your operators in real time.

Features:
* Easy Setup: Just copy and paste your Project ID from the LiveChat Dashboard.
* Simple Integration: The lightweight tracking code is automatically inserted into your theme's footer.
* No complex setup.

== Installation ==

1. Upload the `livechat-widget` directory to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> LiveChat Widget in the left sidebar.
4. Enter your Project ID from your dashboard, and click Save.
5. Setup the rest of your preferences (Appearance, Working Hours, etc) in your LiveChat Dashboard.

== Frequently Asked Questions ==

= Where do I find my Project ID? =

Log in to the LiveChat Operator Dashboard. Go to Settings -> Install to find your Project ID and installation code. Or simply copy the ID from the URL.
